# Nuvex — Deploy to Vercel in 2 minutes

## Step 1 — Go to vercel.com
Create a free account if you don't have one.

## Step 2 — Upload this folder
- Click **"Add New Project"**
- Drag and drop the entire **nuvex** folder
- Vercel auto-detects it as a Vite/React app

## Step 3 — Deploy
- Click **Deploy** — takes about 60 seconds
- You get a live link like: `nuvex.vercel.app`

## Step 4 — Add to iPhone Home Screen
1. Open the live link in **Safari**
2. Tap the **Share button** (box with arrow up)
3. Tap **"Add to Home Screen"**
4. Tap **Add** — done ✅
